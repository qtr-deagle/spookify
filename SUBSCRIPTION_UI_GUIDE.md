### Orange Primary Colors
```

```

### Gradient Combinations
- Main Gradient: `from-orange-500 to-amber-500`
- Light Overlay: `from-orange-500/10 to-amber-500/10`
- Border: `border-orange-500/30`
- Text: `text-orange-400`, `text-orange-500`

### States
- **Normal**: `from-orange-500/20 to-amber-500/20`
- **Hover**: `from-orange-500/30 to-amber-500/30` or `from-orange-600 to-amber-600`
- **Disabled**: `opacity-50` or muted colors
- **Active/Current**: Green badge for current plan

- All features visible
- Plan cards: 3 columns (scales on Premium)
- Smooth gradients and animations
- Full hover effects
